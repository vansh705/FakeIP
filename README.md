# FakeIP
FakeIP is easy-to-use npm package which generates fake ip!

# Guide
**Step 1:**
First, install fakeip package using NPM:
`npm install fakeip`

**Step 2:**
Now Follow the example code

# Example:
```javascript
const fakeip = require('fakeip');
fakeip("ip")
```
